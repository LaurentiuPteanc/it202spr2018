Answers to questions

  1. The Box Model is the way elements are designed and layed out in CSS. Specifically the way each element is surounded by a box.
  2. A block level element is an HMTL element that uses up an entire box in the Box Model. When rendered, they are seperated from other boxes with new lines.
  3. The p tag.
  4. An inline element is an element that does not get seperated from others by a new line.
  5. The b element.
  6. CSS can be included in the pages by using internal style sheets written in the HTML Head tag or by creating a link to a CSS file and downloading it when the page is rendered.
  7. A CSS selector is used to determine which HTML elements to style in the given style.

  8. body {background-color: #0000FF;}
